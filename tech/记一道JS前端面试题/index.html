输出部分：

将Foo作为对象，调用其getName()成员方法，11 行定义（改写后），输出 2。

调用函数getName()，17 行赋值，输出 4。

此处的输出会由于运行环境而有所不同，首先Foo()作为函数被调用，全局变量getName被赋新值，返回this。在浏览器环境下，this->window，window.getName()，输出 1。在 node 环境下其返回值this->global。注意到 node 中的全局变量不作为全局对象的成员，所以调用global.getName()产生Uncaught TypeError: Foo(...).getName is not a function错误。

getName()在 6 行赋值，输出 1。

Foo.getName()无返回值，new为干扰项，输出 2。

新对象通过原型继承的方法getName()，14 行定义，输出 3。

总结

纵观全部代码，共有 3 处变量名为getName，分别为全局变量，Foo对象的成员方法以及 通过对象构造器Foo()原型继承的方法。只要分清这 3 个同名的变量，输出就迎刃而解了。
